# Repository documentation

## Repository structure

```
.
├── docs                # Documentation entrypoint
├── images              # Contains the Dockerfiles for  the base and standalone images
├── jenkins             # Contains the Jenkins configuration as code as well as the pipeline scripts 
├── kubernetes          # Source code for the kubeapi app binary which is used to interact with the kubernetes cluster
├── scripts             # Contains utility scripts for starting the project
└── whanos_example_apps # A collection of applications that can be used with whanos
```
